NAVITROL USER INSTRUCTIONS - Navitec Systems Oy CONFIDENTIAL

Wall monitoring

Updated 26.01.2024

Overview

Introduction

The wall monitoring feature is used for obstacle detection. A slowdown or stop trigger is activated if the AGV's frame is at a certain distance from the obstacle. For obstacle avoidance, see document Obstacle avoidance.

Limitations

  • Not a safety feature
  • Only applicable to position control machine types
  • Maximum two scanners
  • The AGV frame dimensions cannot be smaller than the difference between slow and stop distance

How to use

All valid scanner points at a distance larger than wallmon_min_scan_dist are evaluated for wall monitoring. The AGV frame is extrapolated in the current AGV frame direction in increments of wallmon_pred_ds. The AGV frame is defined by parameters dim_rear, dim_front, dim_left, dim_right and wallmon_pred_safety_margin. See Figure below. If more points than wallmon_min_hits occupy the AGV frame, the AGV will either slow down or stop depending on the distance to the obstacle.

The AGV frame used for wall monitoring. The black dashed rectangle visualizes the complete AGV frame including the margin.

Set up

Quick start

Required parameters

ParameterRecommended valueDescription
I, wallmon_enabled1Enables wall monitor feature that can be used to stop the AGV based on detected nearby objects.
F, wallmon_slow_dist0.4Slow down if scan points are detected closer than this
F, wallmon_stop_dist0.1Stop if scan points are detected closer than this.

Fine-tuning

Optional parameters

ParameterRecommended valueDescription
F, wallmon_stop_manual_speed0.0Control speed that is used instead of stopping when soft stop, hard stop or emergency stop is active during manual mode.
F, wallmon_pred_ds0.01Step size for wall monitor collision prediction in meters. Lower value takes more processing time but is more accurate.
F, wallmon_pred_safety_margin0.05How close to the AGV's predicted frame the scans must be to trigger slow or stop.
I, wallmon_min_hits3Minimum scanner measurement hits required within wall monitor's stop and slow safety areas to trigger slow or stop.
F, wallmon_min_scan_dist0.01Scanner measurements from shorter distances [m] than this are ignored. Can be used to avoid false positives caused by noise.
