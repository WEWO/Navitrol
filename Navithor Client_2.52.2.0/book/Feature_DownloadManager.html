Introduction

Download Manager is a feature which allows user to download log files and other production files from the server without a need to access server over the file explorer, for instance. Download Manager is embedded inside Navithor Client and can be opened from View -> Download Manager menu bar.

Changes to log saving and downloading functions

Together with Download Manager, some changes for saving and downloading logs were made. Now, when user requests saving logs from the client, all the corresponding log files are gathered and .zip archive is created automatically. Progress bar indicates the log saving process.

If user has sufficient user rights (see Section User rights), prompt will appear and ask if user wants to download log file from the server. By pressing Yes, Download Manager form is launched, and downloading of the log file is started automatically. Progress can be seen in progress bar and with the transferred bytes.

Without sufficient user rights, the prompt will just indicate about a successful or unsuccessful saving.

Download Manager

Download Manager main screen.

Download Manager can be used to download log files or other production files, such as statistics, from the server. With sufficient user rights (see chapter User rights), Download Manager can be opened from View -> Download manager menu bar. With a normal access, user can see log file listing of server log folder. Listed are automatically saved log files and user saved log files. Log file can be selected and downloaded to local folder which is specified. After changing the local destination folder, client remembers the selection and same folder will be used next time. By pressing Download button, download starts. Progress bar and transferred bytes are indicating the progress of the transfer. A prompt will appear after successful or unsuccessful transfer.

Advanced features

When user is granted with ManageServerLogFiles user right (see next Chapter), advanced features will be enabled. The dropdown list will appear, and user is able to select different file types from the list. Available types are

NamePurpose
Log ArchiveAuto saved and User saved log file archives.
Binary LogsCurrent and archived binary log files from BinaryLogs folder in server.
Current log filesAll TC parameter files, NavithorLog files, startInfo and audit logs. Current Navithor.log file is not available as it is locked by the server.
MES LogsAll MES log files from MesLogs folder in server.
Production area filesProduction area files from server log folder.
StatisticsProduction statistic files from ProductionData folder in server.

All different file types can be downloaded just as normal log archives.

User rights

For Download Manager two new user rights have been added:

NamePurpose
ViewDownloadManagerUser is able to open Download Manager and download logs
ManageServerLogFilesUser is able to open Download Manager with advanced access to all server log files and production files

If user is granted rights for ManageServerLogFiles, system will automatically grant right for ViewDownloadManager as well.
