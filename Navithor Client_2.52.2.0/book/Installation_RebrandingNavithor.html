Rebranding

This document contains information on "How to rebrand Navithor.

Introduction

This document explains how to rebrand Navithor in order to add a new logo on the application, a new icon and a new name for Navithor Client & Server.

Navithor Client rebranding

Changing icons and company logo

Icons are loaded from "Navithor client directory/Icons". All of these can be replaced with new icons with same size.

Company logo is called "CLogo.png" which can be either removed (no logo will be printed) or replaced.

Navithor Client executable icon can be changed by adding "Client.ico" to same directory as "Navithor Client.exe".

Changing name

NavithorClient.exe.config has a setting named ApplicationName. Setting the value will change the name of "Navithor Client" to the defined application name.

Navithor Server rebranding

Changing icon

Folder where Navithor server is located can have a file named Server.ico. If it exists it is used instead of the default Navitec icon.

Changing name

NavithorServer.exe.config has a setting named ApplicationName. Setting the value will change the name of "Navithor Server" to the defined application name.

Changing server logo

Navithor Server logo can be changed by adding a file named Logo.png in the server folder.

Navithor Web Client rebranding

Logo and icon change

When Web Client is included, Navithor Server installation folder will contain static folder. This folder contains three .png image files. These files can be replaced with new files with same names and dimensions. The files are:

  • cropped-favicon-32x32.png Favicon logo for the web browser
  • logo-white.png Logo shown in the navigation menu
  • logo.png Logo shown in the login view
