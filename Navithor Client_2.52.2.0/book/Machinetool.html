Introduction

Machine Tool is used to create xml-type parameter files for different kinds of machines. Created parameters are then common for all AGVs of the same type. The parameters contain information about following data:

  • Machine dimensions

  • Machine kinematic model

  • Obstacle free zone around the AGV

  • Battery information

  • Loading/unloading capability

  • Data configurations for AGVs input data

  • Resource types available

  • How resource types are rendered

The parameter file is by default named as "parameters_machineTypeName_machineTypeID_version.xml". This file is used by Navithor Server and Navithor Tools. Machine Tool can be started from Servers main menu selection Tools->Machine Tool or as standalone application if provided.

Figure : Main view of Machine Tool. Red rectangle around the vehicle shows the obstacle free zone.

Starting from version 1.9, Machine Tool has been extended to support multiple resource types. Available resource types can be configured by clicking "Edit resource types" button. Opened window (Figure 7. Modify resource types window) shows all available resource types for this machine type. All resource types listed here will be embedded into parameter file upon saving. Manual id field allows configuring which id this resource type will have, leaving the default -1 causes id to be autogenerated. Id 520 is reserved for unknown resource type.

Figure : Modify resource types window

On left side, there is button "Edit load limits", which allows configuring how many of each resource type machines with this machine type are allowed to carry at once.

Figure : Edit load limits window

On right side, there are options to configure how resource type should be rendered. These settings are purely visual.

Figure : Resource type rendering options

  • Type: Changes which resource type rendering is being configured

  • Color: Changes color for resource type

  • Width: How wide resource is

  • Length: How long resource is

  • X Offset: Changes where resource center point is along x-axis

  • Y Offset: Changes where resource center point is along y-axis

  • Z Rotation: Changes rotation of z-axis (x,y plane rotation)s
